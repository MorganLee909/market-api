API Docs for Market

Routes

GET:/api/vendor/{{Vendor ID}}

Retrieve the data for a single Vendor. Private data will only be included if data is for the currently logged in user.

response = Vendor

POST:/api/vendor

Creates a new vendor

body = {
    name (String): vendor name
    email (String): vendor email
    password (String): vendor password
    confirmPassword (String): confirmation password
    description (String): vendor description
}
response = Vendor

PUT:/api/vendor

Update vendor profile data

body = {
    vendor (String, required): vendor ID
    name (String, optional): new vendor name
    email (String, optional): new vendor email
    description (String optional): new vendor description
}
response = Vendor

POST:/api/vendor/login

Vendor login route

body = {
    email (String): vendor email
    password (String): vendor password
}
response = Vendor

GET:/api/vendor/logout

Vendor logout route

response = {}

Vendor
